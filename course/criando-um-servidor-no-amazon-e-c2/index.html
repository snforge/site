DISPONÍVEL NO NÍVEL GRATUITO DA AWS*As instâncias T2 são um tipo de instância do Amazon EC2 projetado para reduzir consideravelmente os custos de aplicativos que se beneficiam da capacidade de usar toda a performance do núcleo de forma intermitente, sempre que necessário. As instâncias T2 estão disponíveis para uso no nível gratuito da AWS, que inclui 750 horas de instâncias t2.micro para Linux e Windows por mês durante um ano para novos clientes da AWS.PERFORMANCE DA CPU COM CAPACIDADE DE INTERMITÊNCIAAs instâncias T2 são instâncias de performance com capacidade de intermitência que oferecem uma linha de base de performance de CPU com capacidade de intermitência acima dessa linha. A performance da linha de base e a capacidade de intermitência são controladas pelos créditos de CPU. As instâncias T2 acumulam créditos de CPU quando estão ociosas e usam esses créditos quando estão ativas.T2 ILIMITADAAs instâncias T2 ilimitadas podem manter a alta performance de CPU pelo tempo necessário para a carga de trabalho. Para a maioria das cargas de trabalho de uso geral, as instâncias T2 ilimitadas oferecerão performance ampla, sem cobranças adicionais. Se a instância precisar ser executada com uma utilização maior de CPU por um período prolongado, ela também poderá fazer isso mediante uma tarifa fixa adicional de 5 centavos por hora de vCPU.PERFORMANCE DE EBS COM CAPACIDADE DE INTERMITÊNCIAAs instâncias T2 funcionam bem com volumes de uso geral (SSD) do Amazon EBS para armazenamento de blocos de instâncias. Os volumes de uso geral (SSD) são o tipo de volume padrão do EBS para instâncias do Amazon EC2. Oferecem capacidade de intermitência de 3.000 IOPS por volume, independentemente do tamanho do volume, para atender às necessidades de performance da maioria dos aplicativos. Além disso, também fornecem uma linha de base consistente de 3 IOPS por GB.
